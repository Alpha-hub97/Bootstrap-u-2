Spinners are mainly used to: Improve user experience (UX) Indicate that an operation is in progress Prevent users from repeatedly clicking buttons Reduce confusion during loading times

Cards in Coding – Report 1. Introduction
In coding, a card is a user interface (UI) component used to display information in a clean, organized, and visually separated way. Cards group related content such as text, images, buttons, or links into a single container. Cards are very common in websites, mobile apps, and dashboards. 2. Purpose of Cards

Cards are used to:
Organize information clearly Improve readability Make interfaces visually attractive Help users scan content quickly Present repeatable content layouts

Pagination in Coding – Report 1. Introduction

In coding, pagination is a technique used to divide large amounts of data into smaller pages. Instead of showing all data at once, content is split into pages that users can navigate using page numbers, “Next”, or “Previous” buttons. Pagination is commonly used in websites, apps, and databases. 2. Purpose of Pagination

Pagination is used to:
Improve website and app performance Reduce loading time Make data easier to read Improve user experience Manage large datasets efficiently
